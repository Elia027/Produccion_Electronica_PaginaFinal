Instalación y activación

Pasos para descargar, iniciar sesión y activar la licencia.

Paso 1: Ve a la página de Altium oficial Descargas de Altium en la selección de descargar y dar clic en donde dice "Online Installer (exe, 29.68 MB)" de ahí solo aceptar los términos y esperar a que se descargue a tu dispositivo. (img 1)
Paso 2: Una vez estando aquí iniciar sesión con la cuenta institucional que te proporcionen y activar la primera licencia. (img 2)

Crear un nuevo esquemático y guardado

Cómo iniciar un esquemático y garantizar que el documento quede guardado correctamente.

Paso 3: En el lado superior izquierdo le darás a "File → New → Schematic" con el fin de crear tu esquemático, sin embargo trata de guardar todo cada cierto tiempo. (img 3)

Barra de menús — visión general

La barra de menús organiza y da acceso a la mayoría de funciones del programa.

La barra de menús es un elemento gráfico fundamental de la interfaz de usuario de un programa (GUI - Graphical User Interface). Se ubica típicamente en la parte superior de la ventana, justo debajo de la barra de título, y consiste en una fila horizontal de palabras o categorías.

Su función principal es organizar y dar acceso a todas o la mayoría de las funciones y comandos del software, agrupándolos de forma lógica y jerárquica bajo títulos generales. (img 7)

img 7Barra de menús

Menú File

Funciones principales relacionadas con creación, apertura, guardado e intercambio de documentos. (img 8)

img 8Opciones del menú File

SECCIÓN: "New"

New

Abre un submenú para crear nuevos documentos individuales (esquemático, PCB, biblioteca, etc.), pero sin un proyecto que los contenga.

Open... (Ctrl+O)

Abre un documento o proyecto existente desde tu computadora.

Ctrl+O

Close (Ctrl+F4)

Cierra el documento activo actualmente en la ventana de trabajo.

Ctrl+F4

SECCIÓN: "Open Project..."

Open Project...

Abre un archivo de proyecto de Altium (.PrjPcb, .PrjCor, etc.) con todos sus documentos asociados.

Open Project Group...

Abre un grupo de proyectos (.DsnWrk), que permite trabajar con varios proyectos interconectados a la vez (muy útil para sistemas complejos).

SECCIÓN: "Save" (Guardar)

Save (Ctrl+S)

Guarda el documento actualmente activo.

Ctrl+S

Save As Managed Sheet to Server...

Guarda una hoja de esquemático en un servidor administrado (Altium 365 o similar), para trabajo colaborativo en la nube.

Save As...

Guarda el documento activo con un nuevo nombre o formato.

Save Copy As...

Crea una copia independiente del documento activo, sin cambiar la referencia al original.

Save All

Guarda todos los documentos abiertos en el proyecto actual.

Save Project As...

Guarda el proyecto completo con un nuevo nombre.

Save Project Group As...

Guarda el grupo de proyectos con un nuevo nombre.

SECCIÓN: "Import" / "Export"

Import

Abre un submenú para importar archivos de otros formatos (Ej.: DXF, Eagle, OrCAD, etc.) a documentos de Altium.

Export

Submenú para exportar documentos de Altium a otros formatos (STEP, PDF, DXF, etc.).

Import Wizard...

Un asistente paso a paso para importar proyectos completos desde otros softwares de PCB (como Eagle, OrCAD, PADS) a Altium.

Run Script...

Ejecuta scripts (en Delphi, VB, etc.) para automatizar tareas dentro de Altium.

SECCIÓN: "Library Importer"

Library Importer

Herramienta específica para importar bibliotecas de componentes desde otros sistemas (como Eagle, OrCAD libraries) a las bibliotecas integradas de Altium.

SECCIÓN: "Print" / "Smart PDF"

Print... (Ctrl+P)

Abre el cuadro de diálogo para imprimir el documento activo (esquemático, plano de PCB, etc.).

Ctrl+P

Smart PDF...

Crea un PDF inteligente del proyecto. No solo es una imagen, sino que puede incluir enlaces clicables entre símbolos, huellas y pistas, muy útil para revisión y documentación.

SECCIÓN: "Recent Documents/Projects/Groups"

Recent Documents

Lista de los documentos individuales abiertos recientemente.

Recent Projects

Lista de los proyectos abiertos recientemente.

Recent Project Groups

Lista de los grupos de proyectos usados recientemente.

SECCIÓN FINAL: "Exit"

Exit (Alt+F4)

Cierra Altium Designer por completo. Si hay trabajo sin guardar, te preguntará si deseas guardarlo antes de salir.

Alt+F4

Resumen de la lógica del menú File: Parte superior (New, Open, Close): Para iniciar trabajo con documentos sueltos. Parte central (Save, Import/Export): Para gestionar la persistencia (guardar, convertir formatos). Parte inferior (Print, PDF, Recent, Exit): Para salida, acceso rápido y cierre. Es el menú desde donde gestionas el ciclo de vida completo de tus archivos en Altium, desde que los creas hasta que los exportas o cierras el programa.

Menú Edit

Operaciones para editar, buscar, seleccionar y organizar objetos en el documento. (img 9)

img 9Vista del menú Edit

Agrupa todas las operaciones para deshacer cambios, manipular objetos (cortar/copiar/pegar), buscar, seleccionar, modificar y organizar elementos en la hoja de trabajo.

SECCIÓN: "Undo / Redo / Clipboard" (Portapapeles)

Undo (Ctrl+Z)

Deshace la última acción realizada. Es tu salvavidas ante errores.

Ctrl+Z

Redo (Ctrl+Y)

Rehace la última acción que fue deshecha. El texto "Nothing to Redo" cambia a "Redo" cuando hay algo para rehacer.

Ctrl+Y

Cut (Ctrl+X)

Corta los objetos seleccionados y los coloca en el portapapeles.

Ctrl+X

Copy (Ctrl+C)

Copia los objetos seleccionados al portapapeles.

Ctrl+C

Paste (Ctrl+V)

Pega el contenido del portapapeles en la ubicación del cursor.

Ctrl+V

Smart Paste... (Shift+Ctrl+V)

Pega inteligente. Te permite pegar elementos del portapapés y elegir cómo se integran (creando nuevas etiquetas de red automáticamente, por ejemplo). Muy potente.

Shift+Ctrl+V

SECCIÓN: "Find & Replace" (Buscar y Reemplazar)

Find Text... (Ctrl+F)

Busca texto específico (nombres de componentes, etiquetas de red, valores) en el documento activo.

Ctrl+F

Replace Text... (Ctrl+H)

Busca un texto y lo reemplaza por otro en el documento. Útil para renombrar masivamente nets o componentes.

Ctrl+H

Find Next (F3)

Repite la última búsqueda para saltar a la siguiente coincidencia.

F3

SECCIÓN: "Select" / "DeSelect" (Selección)

Select

Abre un submenú para seleccionar objetos de formas específicas: Inside Area, Outside Area, All, Connection, Similar Objects, etc.

DeSelect

Abre un submenú para deseleccionar objetos de formas específicas: Inside Area, Outside Area, All, etc.

SECCIÓN: "Delete" / "Break Wire" (Eliminar)

Delete

Elimina permanentemente los objetos seleccionados (tecla Supr o Del también funciona).

Break Wire

Rompe un segmento de cable (wire) o trazado (track) en el punto donde hagas clic, dividiéndolo en dos segmentos. Herramienta esencial en esquemáticos y PCB.

SECCIÓN: "Duplicate" / "Move" / "Align" (Duplicar y Organizar)

Duplicate (Ctrl+R)

Crea una copia instantánea de la selección y la deja lista para colocar, sin usar el portapapeles. Muy eficiente.

Ctrl+R

Move

Submenú para mover objetos de formas especiales: Move, Drag, Move Selection, Move by X, Y... (mover una distancia exacta).

Align

Submenú con herramientas para alinear objetos seleccionados entre sí: Align Left, Align Right, Align Top, Distribute Horizontally, etc. Es clave para un diseño ordenado.

SECCIÓN: "Jump" (Saltar)

Jump

Submenú para mover rápidamente la vista a ubicaciones específicas: Absolute Origin (origen absoluto), Current Origin, Location... (coordenada X,Y exacta), Component... (saltar a un componente por su designador).

SECCIÓN: "Selection Memory" (Memoria de Selección)

Selection Memory

Permite guardar hasta 8 selecciones de objetos en "memoria" (con las teclas Ctrl+1 a Ctrl+8) y recuperarlas después (con las teclas 1 a 8). Imprescindible para trabajar eficientemente en PCB complejos.

SECCIÓN: "Increment Part Number"

Increment Part Number

Incrementa automáticamente el número de parte (designador, como R1, R2, C1, U1) al colocar múltiples copias del mismo componente. Acelera la colocación.

SECCIÓN: "Find Similar Objects"

Find Similar Objects (Shift+F)

La herramienta de poder de Altium. Al hacer clic sobre un objeto, abre un cuadro de diálogo que te permite seleccionar todos los objetos en el documento con propiedades similares y luego editarlos todos a la vez desde el panel Properties. Es la forma más rápida de hacer cambios globales.

Shift+F

Resumen de la lógica del menú Edit: Parte superior (Undo, Clipboard): Control básico y manipulación esencial de objetos. Parte central (Find, Select, Delete): Herramientas de edición y localización para gestionar lo que hay en tu diseño. Parte inferior (Duplicate, Align, Jump, Memory, Find Similar): Herramientas avanzadas de productividad para organizar, navegar y modificar masivamente con precisión y velocidad. En esencia, el menú Edit es tu caja de herramientas para dar forma y modificar tu diseño una vez has colocado los elementos iniciales.

Menú View

Controla la visualización: zoom, paneles, grids y presentación de nets. (img 10)

img 10Paneles y vista (View)

Controla el zoom, el área visible, la visualización de elementos auxiliares (como barras de herramientas y paneles), la cuadrícula (grid) y efectos visuales especiales (como colores para nets).

SECCIÓN: "Zoom y Enfoque" (Control del Área Visible)

Fit Document

Ajusta la vista para que se vea toda la hoja de documento (incluyendo sus bordes). Útil para ver los límites.

Fit All Objects (Ctrl+PgDn)

Ajusta el zoom para que se vean todos los objetos colocados en el documento activo. Es el más usado, ya que ignora el área vacía.

Ctrl+PgDn

Area

Permite dibujar un rectángulo con el ratón para hacer zoom a esa área específica.

Around Point

Similar a "Area", pero define el área haciendo clic para el centro y luego arrastrando.

Selected Objects

Ajusta el zoom para encuadrar perfectamente los objetos que tienes seleccionados.

Zoom In (PgUp)

Acerca la vista (aumenta el zoom) centrado en el cursor.

PgUp

Zoom Out (PgDn)

Aleja la vista (disminuye el zoom) centrado en el cursor.

PgDn

Zoom Last

Vuelve al nivel de zoom anterior. Es un "deshacer" para el zoom.

Full Screen (Alt+F5)

Activa/desactiva el modo de pantalla completa, ocultando barras de título y menús para maximizar el área de trabajo.

Alt+F5

SECCIÓN: "Visualización de Nets (Redes)"

Show Net Color Override (F5)

Activa/desactiva la visualización de los colores personalizados asignados a las redes (nets). Cuando está activo, las pistas y pads de una net se muestran con el color que le hayas asignado (muy útil para rastrear señales críticas como relojes, alimentaciones, etc.).

F5

Set Net Colors

Te lleva a la función para asignar manualmente un color a una red específica (normalmente se hace desde el panel PCB o SCH).

SECCIÓN: "Interfaz de Usuario" (Barras y Paneles)

Toolbars

Submenú para mostrar u ocultar las distintas barras de herramientas (como PCB Standard, Wiring, Utilities, etc.). Te permite personalizar tu espacio de trabajo.

Panels

El comando más importante para acceder a las funcionalidades de Altium. Abre una lista de todos los paneles disponibles (como Properties, PCB, SCH, Libraries, Projects, etc.). Como los paneles se cierran a menudo, desde aquí los puedes reabrir.

SECCIÓN: "Grids y Unidades" (Cuadrícula y Medidas)

Grids

Submenú para configurar la visibilidad y el ajuste (snap) de la cuadrícula. Incluye: Toggle Visible Grid (ver/ocultar los puntos/líneas), Toggle Snap Grid (activar/desactivar el "imán" que hace que los objetos se alineen a la grid), Set Snap Grid... (definir el paso de la grid, ej. 1mm, 10mil, 0.1mm).

Toggle Units

Cambia entre las unidades métricas (mm) e imperiales (mils) para todo el documento activo. Es un interruptor rápido muy útil (Altium usa mils por defecto, 1 mil = 0.001 pulgada).

Resumen de la lógica del menú View: Parte superior (Zoom): Navegación espacial. Te mueves por el documento para acercarte a donde necesitas trabajar. Parte central (Net Colors): Énfasis visual. Destacas elementos críticos para el diseño y la depuración. Parte inferior (Toolbars, Panels, Grids): Personalización del entorno. Configuras qué herramientas ves y cómo interactúas con el espacio de trabajo (grid y unidades). En esencia, el menú View es tu control remoto para la pantalla.

Menú Project

Gestión, validación, versión y empaquetado del proyecto. (img 11)

img 11Gestión de proyecto

Se enfoca en la gestión de archivos dentro del proyecto, validación del diseño, control de versiones, empaquetado y configuración global de las reglas y opciones del proyecto.

SECCIÓN: "Validate" (Validar)

Validate

Ejecuta las verificaciones de reglas eléctricas (ERC - Electrical Rule Check) para el esquemático y las reglas de diseño (DRC - Design Rule Check) para el PCB (dependiendo del documento activo). Es el comando para corregir errores antes de fabricar. Identifica problemas como pines sin conectar, conflictos de alimentación, etc.

SECCIÓN: "Add/Remove Documents" (Añadir/Eliminar)

Add New to Project

Abre un submenú para crear y añadir un nuevo documento en blanco (Esquemático, PCB, Biblioteca, etc.) directamente dentro del proyecto actual.

Add Existing to Project...

Te permite buscar y agregar un documento existente (un esquemático, un PCB, un archivo de simulación, etc.) desde tu disco duro al proyecto actual.

Remove from Project...

Elimina el documento activo (o los seleccionados en el panel Projects) del proyecto, pero NO los borra del disco. Simplemente los desvincula.

Project Documents... (Ctrl+Alt+O)

Abre un cuadro de diálogo con la lista de todos los documentos del proyecto, permitiendo gestionarlos (abrir, cerrar, cambiar orden, etc.) de una vez.

Ctrl+Alt+O

SECCIÓN: "Close Project Documents"

Close Project Documents

Cierra todos los documentos abiertos que pertenecen al proyecto actual. Es una forma rápida de limpiar el espacio de trabajo sin cerrar el proyecto en sí.

SECCIÓN: "Differences" (Comparación)

Show Differences...

Compara dos documentos o proyectos y muestra las diferencias en un panel especial. Muy útil para comparar una revisión de un PCB con otra o ver qué cambió entre el esquemático y el PCB.

Show Physical Differences...

Una comparación más visual y gráfica, que resalta las diferencias físicas (como formas de cobre o trazados) superponiendo colores.

SECCIÓN: "History & Version Control" (Control de Versiones)

History & Version Control

Submenú para integrar con sistemas de control de versiones (como Git, SVN, o el propio Altium 365). Permite Commit, Update, View History. Permite trabajar en equipo, mantener un historial de cambios y revertir a versiones anteriores.

SECCIÓN: "Project Packager" (Empaquetado)

Project Packager...

La herramienta para enviar tu proyecto a fabricación o a un compañero. Crea una carpeta comprimida (ZIP) que contiene todos los archivos necesarios (documentos de proyecto, bibliotecas, outputs, etc.), asegurando que nada falte. Es el paso final antes de compartir.

SECCIÓN: "Project Options" (Configuración)

Project Options...

Abre el corazón de la configuración del proyecto. Aquí defines las reglas y parámetros globales que afectan a todo el diseño: Error Reporting, Connection Matrix, Default Print/Netlist/Simulation Settings, Parameters, Search Paths, etc.

Resumen de la lógica del menú Project: Parte superior (Validate): Aseguramiento de calidad. Verifica que tu diseño no tiene errores críticos. Parte central (Add/Remove, Close, Differences): Gestión de la estructura. Construyes y organizas la lista de archivos que componen el proyecto y los comparas. Parte inferior (Version Control, Packager, Options): Gestión de ciclo de vida y configuración. Trabajas en equipo, preparas la salida final y defines las reglas del juego para todo el proyecto. Los comandos más críticos aquí son Validate (para corregir errores) y Project Options (para configurar las reglas).

Menú Place

Inserción de partes, wires, símbolos jerárquicos, texto y herramientas de dibujo. (img 12)

img 12Comandos del menú Place

Contiene todos los comandos para insertar los elementos fundamentales de un diseño electrónico: componentes, conexiones eléctricas, símbolos de organización, anotaciones y formas de dibujo.

SECCIÓN: "Componentes y Conexiones Eléctricas Principales"

Part...

El comando más importante. Abre el panel Libraries para buscar y colocar un componente (resistencias, microcontroladores, conectores) desde tus bibliotecas en el esquemático. Su atajo es esencial (generalmente P, P).

Power Port

Coloca un símbolo de conexión de potencia global (GND, VCC, +5V, -12V). Define nets de alimentación que estarán disponibles en toda la hoja/proyecto.

Wire (Ctrl+W)

Activa la herramienta para dibujar cables (wires). Los wires son las conexiones eléctricas reales en el esquemático. Sin un wire, dos pines no están conectados, aunque estén pegados.

Ctrl+W

SECCIÓN: "Agrupación y Jerarquía de Conexiones (Buses)"

Bus

Dibuja una línea de bus. Un bus es una representación gráfica de un grupo de señales relacionadas (ej., D[0..7], ADDR[0..15]). Es una línea gruesa.

Bus Entry

Coloca las líneas diagonales pequeñas que conectan los wires individuales a la línea de bus.

Net Label

Coloca una etiqueta de red (net label). Es crucial: asigna un nombre a una red (un wire). Cualquier wire o pin con la misma etiqueta de red estará conectado eléctricamente, sin necesidad de dibujar un wire físico entre ellos. Organiza esquemáticos complejos.

SECCIÓN: "Organización de Proyectos Multihojas"

Sheet Symbol

Coloca un símbolo de hoja que representa a otro documento esquemático (.SchDoc) dentro del mismo proyecto. Crea diseños jerárquicos.

Sheet Entry

Coloca una entrada/salida en un Sheet Symbol. Define las señales que entran o salen de esa sub-hoja.

Port

Similar a Net Label, pero específico para conexiones entre hojas en diseños flat (planos). Un Port en una hoja se conecta con un Port del mismo nombre en otra.

Off Sheet Connector

Similar a Port, pero con un símbolo diferente. Usado para conectar esquemáticos de una manera más gráfica, a menudo en diseños multihojas de estilo "flat".

SECCIÓN: "Reutilización y Harnesses"

Device Sheet Symbol

Coloca un símbolo que hace referencia a una "Device Sheet" (hoja de dispositivo), que es un esquemático reutilizable almacenado en una biblioteca central, ideal para circuitos estándar (fuentes de alimentación, entradas USB).

Reuse Block...

Herramienta avanzada para crear un bloque de circuito reutilizable parametrizado, que puede instanciarse múltiples veces. Muy potente para diseños modulares.

Harness

Conector de arnés. Permite agrupar diferentes tipos de señales (señales, buses, otros harnesses) en un solo objeto lógico para una mejor organización en esquemáticos muy complejos.

SECCIÓN: "Directivas e Instrucciones Especiales"

Directives

Submenú para colocar símbolos de directiva que agregan información extra al diseño, que no es parte del circuito eléctrico en sí, pero que instruye a otras herramientas. Ejemplos: No ERC, PCB Layout, Test Point, Differential Pair.

SECCIÓN: "Texto y Anotaciones"

Text String

Coloca una cadena de texto libre (como comentarios, títulos, notas) en el documento. Atajo común: P, T.

Text Frame

Coloca un marco de texto rectangular que puede contener múltiples líneas de texto, útil para bloques de descripción.

Note

Coloca una nota adhesiva (sticky note) visual en el documento.

SECCIÓN: "Herramientas de Dibujo"

Drawing Tools

Submenú con herramientas para dibujar formas no eléctricas (líneas, arcos, polígonos, rectángulos, elipses). Se usan para agregar información gráfica, marcas, áreas de keepout (en PCB) o diagramas de bloques en la documentación.

Resumen de la lógica del menú Place: Parte superior (Part, Wire, Power): Los cimientos. Pones los componentes y los conectas eléctricamente. Parte central (Bus, Net Label, Sheet Symbols): Organización y abstracción. Agrupas señales, usas conectores lógicos y estructuras jerárquicas para manejar la complejidad. Parte inferior (Directives, Text, Drawing): Información y anotación. Añades metadatos, reglas, comentarios y gráficos para completar el diseño y la documentación. Los comandos más usados son Part..., Wire (Ctrl+W), y Net Label.

Menú Design

Generación de netlists, plantillas, sincronización y utilidades de síntesis. (img 13)

img 13Menú Design

Contiene comandos para generar elementos a partir del diseño, sincronizar información entre el esquemático y el PCB, crear netlists, gestionar plantillas y realizar operaciones de síntesis de bibliotecas o documentos.

SECCIÓN: "Import" (Importar)

Import Schematic...

Suele referirse a importar un esquemático de otro software (como OrCAD, Eagle) al formato nativo de Altium. O puede ser para importar cambios desde un PCB.

SECCIÓN: "Make Library" (Crear Bibliotecas)

Make Schematic Library

Crea automáticamente una biblioteca de esquemáticos (.SchLib) que contiene todos los componentes usados en el proyecto actual. Es vital para portabilidad y respaldo de los símbolos que realmente usaste.

Make Integrated Library

Compila y genera una biblioteca integrada (.IntLib) a partir de las bibliotecas de esquemático (.SchLib) y PCB (.PcbLib) referenciadas en tu proyecto. Es el formato final y empaquetado para distribuir tus componentes.

SECCIÓN: "Sheet Templates" (Plantillas de Hoja)

Sheet Templates

Submenú para gestionar plantillas de documento. Permite cargar, guardar o actualizar plantillas (.SchDot, .PcbDot) que definen el formato de título, bordes, logos y parámetros estándar para todas tus hojas de esquemático o PCB, asegurando uniformidad corporativa.

SECCIÓN: "Netlist" (Lista de Conexiones)

Netlist For Project

Genera un archivo de netlist (lista de conexiones) para todo el proyecto. La netlist es el "contrato" o "mapa de conexiones" que se pasa del esquemático al PCB. Es un paso fundamental en el flujo de diseño.

Netlist For Document

Genera una netlist solo para el documento esquemático activo.

SECCIÓN: "Simulate" (Simular)

Simulate

Ejecuta una simulación del circuito (análisis de DC, transitorio, AC, etc.) basada en el esquemático activo. Requiere que los componentes tengan modelos de simulación asociados. Abre el panel Mixed Sim para configurar y ver resultados.

SECCIÓN: "Synchronization & Hierarchy" (Sincronización y Jerarquía)

Create Sheet From Sheet Symbol

Genera automáticamente una nueva hoja esquemática (.SchDoc) en blanco vinculada a un Sheet Symbol que hayas colocado. Es la forma de implementar un diseño jerárquico.

Create Component From Sheet

Crea un símbolo de componente en una biblioteca (.SchLib) a partir de una hoja esquemática seleccionada. Es una forma de convertir un subcircuito en un componente reutilizable.

SECCIÓN: "Rename & Synchronize" (Renombrar y Sincronizar)

Synchronize Sheet Entries and Ports

Sincroniza las Sheet Entries de un Sheet Symbol con los Ports de la hoja esquemática a la que hace referencia. Si añades un nuevo Port en la hoja hija, este comando lo añadirá como Sheet Entry en el símbolo padre (y viceversa). Esencial para mantener la interfaz jerárquica actualizada.

Resumen de la lógica del menú Design: Parte superior (Import, Make Library): Síntesis y conversión. Generas nuevos elementos (bibliotecas) a partir de tu trabajo o importas diseños externos. Parte central (Templates, Netlist, Simulate): Configuración, exportación y verificación. Estableces estándares, generas el mapa de conexiones (netlist) y verificas el comportamiento con simulación. En Altium, el Properties Panel es una herramienta central e interactiva que cambia dinámicamente según lo que tengas seleccionado en el espacio de trabajo, que se encuentra ubicado en una esquina del lado derecho (esquemático, PCB, librería, etc.).

1. Components (img 20)

Esta pestaña muestra propiedades específicas del componente seleccionado en el esquemático. Aquí defines y editas:

  • Designator: Referencia única (ej. R1, U3, C10).
  • Comment: Descripción o valor (ej. 10k, ATMega328P).
  • Description: Texto descriptivo del componente.
  • Footprint: El modelo físico (paquete) que usará en el PCB.
  • Library Link: Información de la librería de origen.
  • Part Choice: Si hay múltiples partes dentro de un componente.
  • Parameters: Parámetros personalizados o técnicos (voltaje, corriente, tolerancia, etc.).

Aquí defines qué es el componente y cómo se vinculará con el PCB.

Esta pestaña está más relacionada con gestión de proyectos, anotaciones y acciones pendientes:

Es una sección más de gestión y revisión, no de propiedades eléctricas.

Ruteado en Altium Designer

Proceso de trazar las rutas o pistas (tracks) que conectan eléctricamente los componentes en una PCB.

img 24Ejemplo de ruteado interactivo en PCB

El ruteado es una de las etapas más críticas del diseño de PCBs en Altium Designer. Consiste en trazar las pistas conductoras de cobre que conectan los pads de los componentes según las reglas definidas en el esquemático.

Tipos de Ruteado

Ruteado Manual

El diseñador traza cada pista manualmente usando herramientas interactivas. Ofrece máximo control para diseños complejos, críticos o de alta frecuencia.

Ctrl+W

Ruteado Automático (Autorouter)

Altium incluye un autorouter que intenta conectar todas las redes automáticamente según reglas predefinidas. Útil para conexiones simples o como punto de partida.

Ruteado Interactivo

Combina lo mejor de ambos. El diseñador guía las rutas críticas y deja que el router complete el resto. Herramientas como Route Guide o ActiveRoute entran en esta categoría.

Herramientas Esenciales de Ruteado

Interactive Routing (Ctrl+W)

Herramienta principal para trazar pistas. Se adapta a reglas de diseño (ancho, separación) y modos de ruteo (ángulos, curvaturas).

Ctrl+W

Differential Pair Routing

Para trazar pares diferenciales (USB, HDMI) con acoplamiento y espaciamiento controlado automáticamente.

Multi-Routing

Permite rutear múltiples redes simultáneamente (ej. buses de datos de 8 o 16 bits).

Fanout

Conecta pads de componentes SMD a vías para distribuir rutas a otras capas. Esencial para BGA y componentes densos.

Reglas de Diseño (Design Rules - DRC)

¡Importante! Sin reglas bien definidas, el ruteado será inviable o defectuoso.

Reglas Principales

  • Width: Ancho de pistas según corriente o impedancia
  • Clearance: Separación eléctrica entre objetos
  • Routing Via Style: Tamaño de vías
  • Differential Pairs: Reglas para pares diferenciales
  • SMD Neckdown: Conexión de pads SMD

Acceso Rápido

  • Atajo: D + R (Design → Rules)
  • Verificación: DRC (Design Rule Check)
  • Panel: PCB Rules and Constraints

Estrategias de Capas

Estrategia Común: Ruteo horizontal en una capa, vertical en la capa adyacente para minimizar cruces.
Multi-capa: Uso inteligente de vías para cambiar entre capas cuando sea necesario.
Señales Críticas: Rutear en capas internas para proteger de interferencias.

Desafíos Comunes y Soluciones

Problemas

  • Congestión: Muchas rutas en poco espacio
  • Señales Críticas: Alta frecuencia, relojes, diferenciales
  • Integridad de Señal: Reflexiones, acoplamientos
  • Disipación Térmica: Pistas para alta corriente

Soluciones

  • Reordenar componentes, usar más vías
  • Rutear primero, control de longitud/impedancia
  • Evitar ángulos agudos, usar terminaciones
  • Pistas anchas, planos de cobre

Flujo de Trabajo Recomendado

1. Preparación: Definir reglas, clases de net, stackup de capas
2. Colocación (Placement): Ubicar componentes óptimamente (80% del éxito del ruteo)
3. Fanout y Rutas Críticas: Conectar componentes densos y rutear señales sensibles
4. Ruteado General: Conectar el resto de las redes, priorizando alimentaciones
5. Revisión: Usar DRC y análisis de integridad de señal

Consejos Prácticos

Atajos y Modos

  • Shift+R: Ciclar modos de obstáculo (Ignore, Push, Walkaround)
  • Ctrl+Click: Finalizar ruta y empezar nueva
  • Space: Cambiar dirección al rutear
  • * (numérico): Cambiar capa y agregar vía

Mejores Prácticas

  • Rutear alimentaciones primero (más anchas)
  • Usar plano de tierra sólido (GND plane)
  • Mantener simetría en pares diferenciales
  • Evitar antenas (stubs) en señales rápidas

Nota: El ruteado en Altium es tanto un arte como una ciencia. Un buen ruteo asegura la funcionalidad, confiabilidad y fabricabilidad de la placa. La práctica constante y el entendimiento de las reglas eléctricas son clave para dominar esta habilidad.

Recuerda: El autorouter no es mágico; un ruteo manual/mediado suele ser necesario para resultados profesionales. La paciencia y atención al detalle marcan la diferencia entre un diseño funcional y uno excelente.

Conclusión y recomendaciones

Consejos prácticos para trabajar con Altium Designer de forma eficiente y profesional.

📋 Mejores Prácticas

  • Guarde con frecuencia: Configure autoguardado y realice commits al control de versiones con regularidad.
  • Use plantillas: Defina plantillas de hoja y bordes corporativos para mantener consistencia entre documentos.
  • Valide continuamente: Ejecute ERC/DRC antes de transferir a PCB o generar outputs para fabricación.
  • Mantenga bibliotecas sincronizadas: Utilice el Footprint/Parameter Manager y actualice desde bibliotecas centrales.

🚀 Flujo de Trabajo Eficiente

  • Aprenda los atajos de teclado principales (Ctrl+S, Ctrl+W, Ctrl+PgDn, etc.)
  • Utilice Cross Probe para navegar entre esquemático y PCB
  • Organice su espacio de trabajo con paneles flotantes
  • Documente cada paso importante del diseño

Trabajo en equipo: Si trabaja en equipo, aproveche Altium 365 o sistemas de control de versiones para colaboración y trazabilidad. Establezca convenciones de nomenclatura y flujos de revisión claros.

Recuerde: Altium Designer es una herramienta poderosa pero compleja. Domine los fundamentos primero (Place, Wire, Net Labels) antes de avanzar a funciones avanzadas (simulación, jerarquías, reglas complejas). La práctica constante y la exploración de las herramientas le permitirán aprovechar al máximo este software profesional de diseño electrónico.
